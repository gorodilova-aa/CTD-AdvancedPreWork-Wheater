# Weather Comparison: North Carolina vs Siberia
An interactive web application that compares 
real-time weather and 7-day forecasts between 
Cary, NC (USA) and Novosibirsk (Russia). 

## How to Run the Project
1. https://github.com/gorodilova-aa/CTD-AdvancedPreWork-Weather
2. Simply open the index.html file in any modern web browser.
Or, if using VS Code, use the Live Server extension for the best experience.

## Features & Requirements
* API Integration: Connects to the Open-Meteo API to retrieve data.
* Multiple Endpoints:
    - Current Weather: Fetches real-time temperature, wind speed, and weather conditions.
    - 7-Day Forecast: Fetches daily max/min temperatures and weather codes.
* Navigation: Includes a navigation bar to switch between current weather and forecast views without reloading the page.
* Dynamic GET Requests: Issues new GET requests every time a navigation button is clicked to ensure data is up to date.
* Responsive Design: Uses Flexbox and CSS "Cards" for a clean, side-by-side comparison.
* Weather Icons: Integrated WMO weather codes to display visual icons for better UX.

## Data Points Used
Cary, NC (USA): Lat 35.79, Lon -78.78
Novosibirsk (Russia): Lat 55.04, Lon 82.93
